# CDN_ProjetoIntegrador5

Criar um modelo para prever e assim, orientar as decisões com base em dados.
Repositório centralizador da documentação gerada na disciplina Projeto Integrador 5 da instituição Fatec Votorantim
 
## 📌 Problemas que queremos resolver
* **Melhorar a gestão de chopeiras:** A empresa dispoẽ de um número limitado de chopeiras, períodos de alta demanda causam a falta de disponibilidade do produto.
* **Necessidade de priorizar clientes:** Clientes apenas pontuais ou que façam a devolução da chopeira somente no último dia do acordado fazem com quem o produto fique indisponível para que outros possam alugar.

## 🚀 Objetivo
- Previsão de vendas: Analisar e projetar demandas sazonais para otimizar o atendimento e entrega
- Predição de estoque: Necessidade de estimar níveis ideais de estoque para evitar faltas ou excesso.
- Classificação de clientes: Classificar a qualidade do cliente com base no perfil, histórico e recorrência. Possibilitando uma ação prioritária ou estratégias de fidelização.
- Previsão de inadimplência de comodato: Definir as chances do atraso na devolução da chopeira.
